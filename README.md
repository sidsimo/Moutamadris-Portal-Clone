A lightweight, pixel-perfect, and fully responsive front-end clone of the official Moroccan educational platform **Moutamadris (Massar)** login portal interface. Developed using semantic HTML5 and clean CSS3 without any external heavy frameworks.

## 🚀 Live Demo
You can view the live project here: https://sidsimo.github.io/principal-massar/style

---

## ✨ Features
* **Pixel Perfect Design:** Accurately replicates the background pattern, colors, card layouts, and typography of the official platform.
* **Fully Responsive:** Smoothly adapts to all screen sizes (Desktops, Tablets, and Mobile phones).
* **Smooth Interactions:** Hover effects on cards for a dynamic user experience.
* **Bilingual Header:** Contains the official Arabic and Amazigh localized headers.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic markup structure.
* **CSS3:** Custom layouts, flexbox positioning, media queries, and radial-gradient styling for texturing.
